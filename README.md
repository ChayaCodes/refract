# Refract

A React-like library for building user interfaces.

## Development

This is a monorepo managed with Lerna.

### Getting Started

```bash
npm install
npm run bootstrap
npm run build
```

## Packages

- `refract` - Core library
- `refract-dom` - DOM renderer
- `refract-test-utils` - Testing utilities
- `babel-preset-refract` - Babel preset
